# Matrices Dot Product Language (ANTLR)

Este repositorio contiene la solución a los numerales:

1) Modelado de gramática de atributos para un lenguaje SQL-CRUD  
2) Diseño de gramática para lenguaje que calcula producto punto entre matrices  
3) Implementación en ANTLR (Python target)

Archivos incluidos:
- MatrixDot.g4
- matrixdot_visitor.py
- run_matrix.py

Ejecutar:
```
antlr4 -Dlanguage=Python3 MatrixDot.g4
python run_matrix.py
```
